## [0.1.0] - 2026-01-10

### Highlights

- Initial release with GitHub repository backend for OmniStorage

### Added

- GitHub backend (`backend/github`) with `ExtendedBackend` implementation
- Read files via GitHub Contents API
- Write files with automatic commit creation
- Batch commits via Git Trees and Commits API for atomic multi-file operations
- Delete files with automatic commit creation
- List files via GitHub Trees API with prefix filtering
- File metadata (size, SHA1 hash) via Stat
- Range read support (offset/limit)
- Personal access token authentication
- GitHub Enterprise support via custom BaseURL/UploadURL
- Configurable commit messages with `{path}` placeholder
- Configurable commit author (name and email)
- Backend registry integration (`github` name)
- Configuration from environment variables
- Configuration from string maps

### Documentation

- README.md with usage examples, batch API, and configuration reference
- ROADMAP.md with planned features
- GoDoc comments on all public APIs

### Tests

- Unit tests for configuration parsing
- Unit tests for path validation and normalization
- Integration tests for backend operations (requires GITHUB_TOKEN)
